# MCP Státusz

## Beállítva
- MCP konfiguráció hozzáadva: `%APPDATA%\Cursor\User\settings.json`
- Backup: `settings.json.backup`

## Cursor újraindítás szükséges
Az MCP szerver csak újraindítás után érhető el.

## Elérhető műveletek (MCP-n keresztül)
- "Listázd a Hostinger VPS-eket"
- "Mutasd a WebSuli VPS információit"

## Korlátozás
Az MCP szerver csak Cursor IDE kontextusában érhető el.
